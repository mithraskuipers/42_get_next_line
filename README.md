### Compiling

Single file descriptor:
`gcc main.c src/get_next_line.c src/get_next_line_utils.c`

Multiple file descriptors:
`gcc main.c src/get_next_line_bonus.c src/get_next_line_utils_bonus.c`

### Executing

Single file descriptor:
`./a.out ./file.txt`

Multiple file descriptors:
`./a.out ./file1.txt ./file2.txt`
